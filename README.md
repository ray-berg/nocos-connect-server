# NOCOS-Connect Server

Self-host your own NOCOS-Connect server for remote desktop connectivity.

## How to build

```bash
cargo build --release
```

Three executables will be generated in target/release.

- hbbs - NOCOS-Connect ID/Rendezvous server
- hbbr - NOCOS-Connect relay server
- nocos-connect-utils - NOCOS-Connect CLI utilities
